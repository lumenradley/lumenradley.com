# Launch Checklist

## Week 1: Foundation

1. Update `astro.config.mjs` with your real production URL.
2. Replace `Your Name` labels in `src/layouts/BaseLayout.astro`.
3. Add your first 5 real posts in `src/content/blog/`.
4. Decide the newsletter path:
   - Buttondown/Substack embed on homepage, or
   - simple external link first.
5. Deploy preview on Vercel/Netlify/Cloudflare Pages.

## Week 2: Distribution

1. Submit `https://yourdomain.com/rss.xml` to feed readers and podcast/newsletter tools.
2. Add social cards (`og:image`) and favicon assets.
3. Add analytics (Plausible or similar privacy-focused provider).
4. Set up redirects for old or syndicated URLs if needed.

## Week 3: Content System

1. Define a writing cadence (for example one deep post per week).
2. Create post templates:
   - essay
   - quick notes
   - link post
3. Add tags and topic pages if categories become important.

## Optional Later

1. Integrate a CMS (Sanity, Contentful, or Decap) if you want browser editing.
2. Add full-text search.
3. Add automated social syndication from newly published posts.
